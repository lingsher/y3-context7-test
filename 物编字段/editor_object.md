# 物编对象 (Editor Object)

Y3 编辑器物编对象字段定义，包含技能、单位、物品、Buff等配置字段。

---

## Object.Ability

| 字段名 | 类型 | 描述 |
| --- | --- | --- |
| ability_break_cast_range | `string[]` | 当技能施法开始或施法出手阶段结束时，如果与施法目标的距离超过该值，会打断技能释放。 |
| ability_build_subtype | `any` | 建造方式 |
| ability_bw_point | `number` | 施法完成时长（也可以叫做技能后摇） |
| ability_cast_point | `number` | 施法开始时长（也可以叫做技能前摇） |
| ability_cast_range | `string[]` | 以角色为圆心，以该值为半径的圆形区域。是角色不需要移动即可释放技能的最远距离。 |
| ability_cast_type | `any` | 技能的释放类型。 |
| ability_channel_time | `number` | 施法出手时长 |
| ability_cost | `string[]` | 释放技能消耗的MP(会根据单位自身的技能资源进行变化)数值 |
| ability_damage | `string[]` | 技能造成的伤害，可使用公式编辑，需在触发内引用才可生效 |
| ability_damage_range | `string[]` | 当前技能的影响范围， |
| ability_hp_cost | `string[]` | 使用该技能时会消耗的生命值 |
| ability_icon | `any` | 技能的图标，会在编辑器和游戏的ui上显示 |
| ability_joystick_check_range | `number` | 摇杆指示器索敌范围 |
| ability_max_level | `number` | 技能的等级上限 |
| ability_max_stack_count | `string[]` | 技能的最大充能数，必须满足充能数大于0且不在冷却状态才能施放该技能 |
| ability_prepare_time | `number` | 施法引导时长 |
| ability_stack_cd | `string[]` | 每增加一层充能数所需的时间（单位为秒） |
| ability_stage_config | `any` | 技能阶段配置 |
| ability_timeline_resource | `any` | 技能绑定动画轨 |
| arrow_length | `string[]` | 箭头指示器的长度 |
| arrow_width | `string[]` | 箭头指示器的宽度 |
| art_resource_btn | `any` | 编辑技能效果 |
| auto_pick | `boolean` | 当采集到的资源是物品时，是否自动将物品拾取。如果不自动拾取则物品会创建在地面。 |
| bs_sfx_list | `any[]` | 技能执行到该阶段时，在自身处显示的特效，技能被打断后会立即停止该特效 |
| bs_sound_effect | `any[]` | 后摇音效 |
| build_list | `any[]` | 建造技能建造的单位类型 |
| build_rotate | `number` | 建造角度 |
| building_attack_range_sfx | `any` | 攻击范围预览的特效，大小参考ID为101492的特效资源 |
| camp_priority_enemy | `number` | 阵营值越小，优先级越高，如果阵营值相同，则距离近优先。 |
| camp_priority_friend | `number` | 阵营值越小，优先级越高，如果阵营值相同，则距离近优先。 |
| camp_priority_neutral | `number` | 阵营值越小，优先级越高，如果阵营值相同，则距离近优先。 |
| camp_priority_self | `number` | 阵营值越小，优先级越高，如果阵营值相同，则距离近优先。 |
| can_autocast_when_attack_target | `boolean` | 勾选后攻击命令能触发自动施法。 |
| can_bs_interrupt | `boolean` | 技能施法完成阶段能否被其他技能或者移动打断 |
| can_cache | `boolean` | 开启时，被控制时，控制结束 依旧可以继续释放（例如被禁止施法时发布施放命令，禁止施法解除后就会自动施放出来） |
| can_cast_interrupt | `boolean` | 技能施法出手阶段能否被其他技能或者移动打断 |
| can_cast_when_hp_insufficient | `boolean` | 如果单位当前生命值不满足消耗时，能否施放技能 |
| can_cost_hp | `boolean` | 开启后可以配置消耗生命值相关的技能参数。如无需要请勿开启该配置，会增加系统消耗。 |
| can_interrupt_others | `boolean` | 释放该技能时候会尝试打断当前正在释放的技能 |
| can_prepare_interrupt | `boolean` | 技能施法引导阶段能否被其他技能或者移动打断 |
| can_ps_interrupt | `boolean` | 技能施法开始阶段能否被其他技能或者移动打断 |
| circle_radius | `string[]` | 圆形指示器的半径 |
| cold_down_time | `string[]` | 技能的冷却时间，释放一次技能后需要等待该时间才可继续释放（单位为秒） |
| collection_animation | `string` | 使用采集技能时会播放的动画。会在技能施法开始时播放，施法停止时停止。 |
| collection_animation_loop | `boolean` | 采集技能时播放的采集动画是否会循环播放。 |
| collection_continuously | `boolean` | 是否在采集完成后继续采集（仅在技能冷却时间为0时生效） |
| collection_destructible_tags | `any` | 技能索敌时只会搜索带有指定标签的可破坏物 |
| cost_hp_can_die | `boolean` | 如果单位当前生命值不满足消耗时施放技能，则该项为true时单位会死亡，为false时会保留1点生命值 |
| cst_sfx_list | `any[]` | 技能执行到该阶段时，在自身处显示的特效，技能被打断后会立即停止该特效 |
| cst_sound_effect | `any[]` | 技能执行到该阶段时，在自身处播放的音效内容，技能被打断后会立即停止该音效 |
| customized_pointer_sfx | `any` | 自定义特效选择 |
| description | `string` | 技能的介绍说明，用在编辑器内和游戏内的Tips显示上 |
| enable_customized_pointer_sfx | `boolean` | 自定义指示器特效 |
| end_sfx_list | `any[]` | 技能执行到该阶段时，在自身处显示的特效，技能被打断后会立即停止该特效 |
| end_sound_effect | `any[]` | 技能执行到该阶段时，在自身处播放的音效内容，技能被打断后会立即停止该音效 |
| filter_condition_camp | `any` | 按阵营选取目标。 |
| filter_condition_move_channel_type | `any` | 技能索敌时只会搜索指定移动类型的单位 |
| filter_condition_type | `any` | 按种类选取目标。 |
| filter_item_tags | `any` | 技能索敌时只会搜索带有指定标签的物品 |
| filter_unit_tags | `any` | 技能索敌时只会搜索带有指定标签的单位 |
| forbid_unit_tags | `any` | 技能索敌时不会搜索带有指定标签的单位 |
| hit_sfx_list | `any[]` | 技能命中时，在命中目标处显示的特效 |
| hit_sound_effect | `any[]` | 技能命中时，会在命中者处播放的音效内容 |
| influenced_by_cd_reduce | `boolean` | 技能受到单位属性中的冷却缩短的影响 |
| influenced_by_move | `boolean` | 勾选后移动会尝试打断当前技能，不勾选则可以实现移动施法 |
| is_autocast | `boolean` | 优先按照阵营选取目标，其次按类别，如果都相同，则距离近优先。 |
| is_charge_ability | `boolean` | 蓄力技能专属，为true时，引导时间会作为蓄力技能一阶段的引导时间。【施法开始阶段需要配置为0】 |
| is_immediate | `any` | 释放这个技能是否需要施法过程，开启时可以在单位存活的任何时期发动该技能，即使被禁止施法也可以正常释放。 |
| is_meele | `boolean` | 开启时，标记这个技能造成的伤害为近战伤害 |
| key | `number` | 技能的ID |
| kv | `any` | 玩家自定义 |
| magicbook_list | `any[]` | 魔法书中存放的技能list |
| name | `string` | 当前技能的名称 |
| need_turn_to_target | `boolean` | 开启时，技能释放后单位会自动转到技能释放的方向（转身速度为单位的转身速度） |
| pick_count | `number` | 每次使用采集技能时获取到的资源数量。如果采集的是玩家属性，则获得对应的玩家属性值，如果采集的是物品，则获得对应数量的物品。 |
| player_props_cost | `any[]` | 施放技能消耗的玩家属性，技能拥有者的玩家该属性不足时无法施放技能 |
| pointer_can_block | `boolean` | 指示器可被阻挡 |
| pointer_channel | `any` | 指示器类型 |
| pointer_limitation | `any` | 被阻挡类型 |
| precondition_list | `any[]` | 释放技能的前置条件 |
| ps_sfx_list | `any[]` | 技能执行到该阶段时，在自身处显示的特效，技能被打断后会立即停止该特效 |
| ps_sound_effect | `any[]` | 技能执行到该阶段时，在自身处播放的音效内容，技能被打断后会立即停止该音效 |
| release_immediately_out_of_range | `boolean` | 如果技能目标点超出施法范围，会在施法范围内离目标最近的点施放 |
| required_level | `any` | 单位学习该技能所需要的等级 |
| sector_angle | `string[]` | 扇形指示器的角度 |
| sector_radius | `string[]` | 扇形指示器的边长 |
| show_building_attack_range | `boolean` | 开启后，建造时的模型会附带攻击范围预览 |
| sight_type | `any` | 释放技能时的鼠标指示器的样式 |
| sound_event_list | `any[]` | 可以设置触发指定事件时播放的声音 |
| sp_count_down | `boolean` | 开启时，会根据技能的施法时间显示相应的进度条 |
| sp_sfx_list | `any[]` | 技能执行到该阶段时，在自身处显示的特效，技能被打断后会立即停止该特效 |
| sp_sound_effect | `any[]` | 技能执行到该阶段时，在自身处播放的音效内容，技能被打断后会立即停止该音效 |
| suffix | `string` | 只在编辑器中使用，用于区分相同名字的物品。并不会影响到游戏内的显示内容 |
| tags | `any[]` | 用于对技能的分类处理。为技能贴上标签后可以对其进行更方便的关系，例如编写游戏逻辑：所有拥有XX标签的技能等级+1 |
| target_arrow_length | `string[]` | 目标长度 |
| target_arrow_width | `string[]` | 目标宽度 |
| target_attribute | `any` | 配置技能索敌时的特殊筛选规则（如排除自己、允许无敌等） |
| target_circle_radius | `string[]` | 目标圆形半径 |
| target_sector_angle | `string[]` | 目标角度 |
| target_sector_radius | `string[]` | 目标扇形半径 |
| transition_duration | `number` | 过渡时间 |
| transition_start_time | `number` | 过渡起始时间 |
| type_priority_building | `number` | 类别值越小，优先级越高，如果类别值相同，则距离近优先。 |
| type_priority_creature | `number` | 类别值越小，优先级越高，如果类别值相同，则距离近优先。 |
| type_priority_destructible | `number` | 类别值越小，优先级越高，如果类别值相同，则距离近优先。 |
| type_priority_hero | `number` | 类别值越小，优先级越高，如果类别值相同，则距离近优先。 |
| type_priority_item | `number` | 类别值越小，优先级越高，如果类别值相同，则距离近优先。 |
| uid | `string` | 技能的唯一id |
